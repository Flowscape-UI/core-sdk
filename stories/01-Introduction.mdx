# 🎨 Flowscape Core SDK

**Powerful 2D canvas engine built on Konva**

Welcome to the interactive documentation for Flowscape Core SDK! Here you can test all library features in real-time.

## ✨ Key Features

- **🎯 Framework-agnostic** — Works seamlessly with React, Vue, Svelte, Angular, or vanilla JavaScript
- **🧩 Plugin System** — Extensible architecture with ready-to-use plugins
- **📐 Complete Toolset** — Grid, rulers, guides, area selection, alignment guides
- **⌨️ Keyboard Shortcuts** — Ctrl+C/V/X, Delete, Ctrl+G for grouping, Ctrl+Z/Shift+Z for undo/redo
- **🎨 Rich Shapes** — Rectangles, circles, ellipses, text, images, arrows, stars, and more
- **🔄 Transformations** — Rotation, scaling, movement with aspect ratio lock
- **✏️ Inline Editing** — Double-click text nodes to edit directly on canvas
- **🕐 History System** — Full undo/redo support with Ctrl+Z
- **📦 TypeScript-first** — Full typing support out of the box
- **🚀 High Performance** — Handles 1000+ nodes without FPS drops
- **🔧 Addons API** — Extend any component with custom functionality

## 🚀 Quick Start

```typescript
import { CoreEngine, GridPlugin, SelectionPlugin, NodeHotkeysPlugin } from '@flowscape-ui/core-sdk';

// Create engine with plugins
const core = new CoreEngine({
  container: document.getElementById('canvas-container')!,
  width: 1200,
  height: 800,
  plugins: [
    new GridPlugin({ enabled: true }),
    new SelectionPlugin({ dragEnabled: true }),
    new NodeHotkeysPlugin(), // Ctrl+C/V/X, Delete
  ],
});

// Add shapes
const rect = core.nodes.addShape({
  x: 100,
  y: 100,
  width: 200,
  height: 150,
  fill: '#3b82f6',
  cornerRadius: 8,
});

const text = core.nodes.addText({
  x: 120,
  y: 140,
  text: 'Hello Flowscape!',
  fontSize: 24,
  fill: 'white',
});
```

## 🏗️ Architecture

Flowscape Core SDK is built on a modular architecture:

```
┌─────────────────────────────────────┐
│         CoreEngine                  │
│  ┌──────────────────────────────┐   │
│  │     Plugin System            │   │
│  │  - GridPlugin                │   │
│  │  - SelectionPlugin           │   │
│  │  - RulerPlugin               │   │
│  │  - NodeHotkeysPlugin         │   │
│  └──────────────────────────────┘   │
│  ┌──────────────────────────────┐   │
│  │     Node Manager             │   │
│  │  - ShapeNode                 │   │
│  │  - TextNode                  │   │
│  │  - ImageNode                 │   │
│  │  - GroupNode                 │   │
│  └──────────────────────────────┘   │
│  ┌──────────────────────────────┐   │
│  │     Camera Manager           │   │
│  │  - Zoom (Ctrl+Wheel)         │   │
│  │  - Pan (Space+Drag)          │   │
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘
```

## 🎉 What's New in 1.0.4

This release adds new ways to bring external content into the canvas and improves UX during rotation:

- **📋 Paste from Clipboard** — paste external content into the canvas using `ContentFromClipboardPlugin`
  - Images (any browser-supported formats)
  - GIFs
  - Videos (any browser-supported formats)
  - SVG files
  - Text
- **🖱️ Drag & Drop from Desktop** — drag & drop supported file types directly into the canvas (enabled via `ContentFromClipboardPlugin`)
- **🌀 Custom Rotation Cursor** — custom cursor when rotating objects for better UX and precision

## 🎉 What's New in 1.0.3

This release brings major improvements:

- **✨ History System** — Full undo/redo support (Ctrl+Z / Ctrl+Shift+Z)
- **📏 Alignment Guides** — Smart guides appear during movement and resizing
- **✏️ Inline Text Editing** — Double-click text nodes to edit directly
- **🔧 Addons API** — Attach custom functionality to any component
- **📐 Canvas Auto-Resize** — Automatically adjusts to window size changes
- **⚡ Performance** — Optimized to handle 1000+ nodes smoothly
- **🎨 New Layer Shortcuts** — Ctrl+Shift+[ / ] for send to back/bring to front
- **📦 Full TypeScript** — Complete type coverage across all components
