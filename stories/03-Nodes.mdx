# 📦 Nodes

Nodes are the building blocks of your canvas. Flowscape SDK provides various node types for different use cases.

## Node Types

- **Shape** — Rectangles with optional rounded corners
- **Circle** — Perfect circles
- **Ellipse** — Oval shapes
- **Text** — Text elements with formatting
- **Image** — Images from URLs or data
- **SVG** — SVG images (as data URI or URL)
- **Video** — Video nodes (HTMLVideoElement-backed)
- **GIF** — Animated GIF nodes
- **Arc** — Arc segments
- **Arrow** — Arrows with customizable pointers
- **Star** — Star shapes with configurable points
- **Ring** — Ring/donut shapes
- **RegularPolygon** — Regular polygons (triangle, pentagon, hexagon, etc.)
- **Group** — Container for grouping nodes

## Common Node Properties

All nodes share these base properties:

| Property    | Type      | Default | Description         |
| ----------- | --------- | ------- | ------------------- |
| `x`         | `number`  | `0`     | X position          |
| `y`         | `number`  | `0`     | Y position          |
| `rotation`  | `number`  | `0`     | Rotation in degrees |
| `scaleX`    | `number`  | `1`     | Horizontal scale    |
| `scaleY`    | `number`  | `1`     | Vertical scale      |
| `opacity`   | `number`  | `1`     | Opacity (0-1)       |
| `visible`   | `boolean` | `true`  | Visibility          |
| `draggable` | `boolean` | `false` | Can be dragged      |
| `id`        | `string`  | auto    | Unique identifier   |
| `name`      | `string`  | `''`    | Optional name       |

## Image Node

Display images from URLs or data URIs.

`src` can be a local path, a remote URL, or a data URI.

```typescript
const image = core.nodes.addImage({
  x: 500,
  y: 100,
  width: 200,
  height: 150,
  src: '/path/to/image.jpg',
});
```

### Image Options

| Property | Type     | Default | Description  |
| -------- | -------- | ------- | ------------ |
| `src`    | `string` | `''`    | Image source |
| `width`  | `number` | auto    | Width        |
| `height` | `number` | auto    | Height       |

### Image Methods

```typescript
// Change image source
image.setSrc('/new/image.jpg');

// Get current source
const src = image.getSrc();
```

## SVG Node

## Group Node

Container for grouping multiple nodes.

```typescript
// Create group
const group = core.nodes.addGroup({
  x: 400,
  y: 400,
  draggable: true,
});

// Add nodes to group
const circle1 = core.nodes.addCircle({ x: 0, y: 0, radius: 30 });
const circle2 = core.nodes.addCircle({ x: 70, y: 0, radius: 30 });

circle1.getKonvaNode().moveTo(group.getKonvaNode());
circle2.getKonvaNode().moveTo(group.getKonvaNode());
```

### Group Options

| Property    | Type      | Default | Description     |
| ----------- | --------- | ------- | --------------- |
| `draggable` | `boolean` | `false` | Can be dragged  |
| `clip`      | `object`  | `null`  | Clipping region |

## Node Methods

All nodes provide these methods:

```typescript
// Position
node.setPosition({ x: 100, y: 200 });
const pos = node.getPosition();

// Size (for applicable nodes)
node.setSize({ width: 200, height: 150 });
const size = node.getSize();

// Rotation
node.setRotation(45);
const rotation = node.getRotation();

// Scale
node.setScale({ x: 1.5, y: 1.5 });
const scale = node.getScale();

// Opacity
node.setOpacity(0.5);
const opacity = node.getOpacity();

// Visibility
node.setVisible(false);
const visible = node.isVisible();

// ID and Name
const id = node.getId();
node.setName('my-node');
const name = node.getName();

// Get Konva node (for advanced use)
const konvaNode = node.getKonvaNode();

// Destroy node
node.destroy();
```

## Working with Addons

Nodes can be extended with addons for additional functionality:

```typescript
import {
  TextAutoTrimAddon,
  ImageHoverFilterAddon,
  ShapeHoverHighlightAddon,
} from '@flowscape-ui/core-sdk';

// Text auto-trim
const text = core.nodes.addText({ text: 'Hello', fontSize: 24 });
text.addons.add(new TextAutoTrimAddon());

// Image hover filter
const image = core.nodes.addImage({ src: '/image.jpg' });
image.addons.add(new ImageHoverFilterAddon({ mode: 'sepia' }));

// Shape hover highlight
const rect = core.nodes.addShape({ width: 100, height: 100 });
rect.addons.add(
  new ShapeHoverHighlightAddon({
    mode: 'fill',
    fill: 'rgba(59, 130, 246, 0.3)',
  }),
);
```

## Best Practices

1. **Use appropriate node types** — Choose the right node for your use case
2. **Set IDs for important nodes** — Makes finding and managing nodes easier
3. **Group related nodes** — Use groups for better organization
4. **Clean up unused nodes** — Call `destroy()` to free memory
5. **Use addons for behavior** — Extend nodes without subclassing
