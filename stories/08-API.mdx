## CoreEngine

Main engine class that manages the canvas.

### Constructor

```typescript
new CoreEngine(options: CoreEngineOptions)
```

### Properties

| Property   | Type             | Description        |
| ---------- | ---------------- | ------------------ |
| `nodes`    | `NodeManager`    | Node management    |
| `camera`   | `CameraManager`  | Camera control     |
| `plugins`  | `Plugins`        | Plugin management  |
| `eventBus` | `EventBus`       | Event system       |
| `history`  | `HistoryManager` | History management |

### Methods

| Method                  | Returns | Description         |
| ----------------------- | ------- | ------------------- |
| `getStage()`            | `Stage` | Get Konva stage     |
| `getLayer()`            | `Layer` | Get main layer      |
| `resize(width, height)` | `void`  | Resize canvas       |
| `destroy()`             | `void`  | Cleanup and destroy |

## Plugin

Base class for creating plugins.

### Methods to Override

| Method           | Description                   |
| ---------------- | ----------------------------- |
| `onAttach(core)` | Called when plugin is added   |
| `onDetach(core)` | Called when plugin is removed |

### Example

```typescript
class CustomPlugin extends Plugin {
  protected onAttach(core: CoreEngine): void {
    // Initialize
  }

  protected onDetach(core: CoreEngine): void {
    // Cleanup
  }
}
```
